## 需求覆盖率对照表

| 需求点 / 验收标准 | 需求描述 | 覆盖用例名称 |
|---|---|---|
| 列表页导出入口 | 报告编制列表操作栏增加“导出”按钮，受导出权限控制 | 列表页导出报告成功、无导出权限按钮隐藏、越权导出接口被拦截 |
| 详情页导出入口 | 报告编制 ONLYOFFICE 预览详情页增加“导出”按钮，点击触发导出 | ONLYOFFICE详情页导出报告成功、导出按钮位于工具栏并列展示 |
| 导出内容-全部导出 | 导出报告编制正文 + 所有附件 | 导出内容包含正文和所有附件、列表页导出报告成功 |
| 导出文件命名规则 | ZIP 命名 {报告名称}_{导出项数量}_{导出日期}.zip，日期格式 YYYYMMDD，导出项数量 = 正文 1 + 附件数量 | ZIP文件命名规则正确、无附件报告导出仅含正文 |
| 正文按文件类型导出 | 报告正文按正文文件类型（Word/PDF）导出 | 导出内容包含正文和所有附件、PDF类型正文按原类型导出 |
| 附件按原始格式导出 | 附件按原始格式导出，文件名保持原始附件名称 | 附件按原始格式和名称导出、导出内容包含正文和所有附件 |
| ZIP 根目录无子文件夹 | 所有文件打包至同一 ZIP，根目录下不创建子文件夹 | 导出内容包含正文和所有附件、列表页导出报告成功 |
| 导出按钮工具栏位置 | 导出按钮位于 ONLYOFFICE 工具栏，与打印、下载等按钮并列 | 导出按钮位于工具栏并列展示、ONLYOFFICE详情页导出报告成功 |
| 点击导出无弹窗下载 | 点击导出后浏览器自动开始下载，无需额外弹窗确认 | 点击导出无弹窗直接下载、列表页导出报告成功 |
| 导出过程加载提示 | 下载过程中显示“正在导出，请稍候…”加载提示 | 导出过程显示加载提示、导出中禁止重复点击 |
| 自动保存默认下载目录 | 导出完成后浏览器自动保存文件至默认下载目录 | 点击导出无弹窗直接下载 |
| 文档不存在异常 | ONLYOFFICE 文档不存在或无法读取时提示“文档不存在，导出失败” | 文档不存在导出失败 |
| 网络中断异常 | 导出过程中网络中断时提示“导出失败，请重试” | 网络中断导出失败提示 |
| 无权限按钮处理 | 用户无导出权限时导出按钮不显示或置灰 | 无导出权限按钮隐藏 |
| 文档内容异常 | 文档内容为空或格式异常时提示“文档内容异常，导出失败” | 文档内容异常导出失败 |
| 列表与详情权限一致 | 列表页和详情页导出按钮权限保持一致 | 列表页与详情页导出权限一致、无导出权限按钮隐藏 |
| 导出审计日志 | 导出操作记录审计日志，包含操作人、操作时间、方案名称/报告名称、文档名称 | 导出成功记录审计日志 |
| 导出不占用系统存储 | 导出文件直接由前端或后端生成并传输给用户，不占用系统存储空间 | 导出文件不占用系统存储 |
