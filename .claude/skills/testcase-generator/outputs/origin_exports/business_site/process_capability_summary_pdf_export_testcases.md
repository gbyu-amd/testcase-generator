## 需求覆盖率对照表

| 需求点 / 验收标准 | 需求描述 | 覆盖用例名称 |
|---|---|---|
| 导出入口与同步导出 | 工艺能力汇总页面右上角增加导出按钮，点击后同步实时导出 PDF，不转异步任务 | 当前方案导出PDF成功 |
| PDF 正式报告内容 | 导出当前方案下监控项目能力数据，包括 Cpk、Ppk、超规格限百分比、异常批次等 | PDF内容指标展示正确、当前方案导出PDF成功 |
| 分页外完整数据 | 前端收集当前方案所有监控项目数据，包括分页外全部数据，列表每页 50 条不影响导出范围 | 分页外数据导出完整 |
| 视图切换不影响导出 | 连接视图和矩阵视图 Tab 不影响导出功能，导出统一使用列表视图格式 | 矩阵视图导出列表数据 |
| 搜索筛选范围 | 页面支持按监控项目名称模糊搜索，导出应基于当前方案和当前数据范围 | 搜索条件下导出当前范围 |
| 导出中加载与防重复 | 后端生成 PDF 过程中显示“导出中，请稍候...”并禁止再次点击导出 | 导出中禁止重复点击 |
| 数据量上限 | 导出数据量超过 500 条时提示无法一次性导出并要求缩小范围或联系管理员 | 超过500条导出被拦截 |
