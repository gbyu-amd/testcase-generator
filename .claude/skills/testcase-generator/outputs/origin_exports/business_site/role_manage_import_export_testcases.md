## 需求覆盖率对照表

| 需求点 / 验收标准 | 需求描述 | 覆盖用例名称 |
|---|---|---|
| 导入入口与模板下载 | 列表上方导入按钮打开弹窗，支持下载角色管理模板 | 打开导入弹窗展示完整、下载角色导入模板成功、模板字段说明与规则一致 |
| 模板字段 | 模板包含角色名称、角色编码、状态、菜单权限、按钮权限、分配用户、数据权限类型和数据权限 | 下载角色导入模板成功、模板字段说明与规则一致 |
| 导入成功流程 | 合格 Excel 上传、校验通过后导入并刷新角色列表 | 合格角色文件导入成功、全部权限角色导入成功、指定权限产品导入成功 |
| 权限配置迁移 | 导入支持菜单权限、按钮权限、分配用户和产品数据权限跨环境迁移 | 导入后权限配置生效、指定权限产品导入成功 |
| 角色编码唯一性 | 角色编码必须唯一，重复时记录错误行号并整体失败 | 角色编码重复整体失败 |
| 关联用户校验 | 分配用户按姓名(用户账号)填写，用户不存在时记录错误行号 | 分配用户不存在整体失败 |
| 产品权限校验 | 指定数据权限产品按产品名称(产品编号)填写，产品不存在时记录错误行号 | 产品编号不存在整体失败 |
| 菜单与按钮权限校验 | 菜单路径和按钮权限格式必须有效 | 菜单路径无效整体失败、按钮权限格式无效被拦截 |
| 文件格式与模板校验 | 仅支持 .xlsx 或 .xls，表头必须匹配系统模板 | 非Excel文件上传被拦截、表头不匹配模板终止导入 |
| 字段与文件异常 | 状态值、角色编码格式、文件大小和多错误行需正确拦截 | 字段多错误按分号展示、文件超过20M被拦截 |
| 错误报告 | 存在错误时拒绝全部导入并提供 TXT 错误报告 | 角色编码重复整体失败、分配用户不存在整体失败、产品编号不存在整体失败、导入失败报告可下载 |
| 导出勾选角色 | 勾选一个或多个角色后导出 Excel，文件名包含导出项数和时间戳 | 勾选角色导出成功 |
| 导出格式 | 导出菜单权限和按钮权限格式与导入模板一致，可修改后再次导入 | 导出权限格式符合模板、导出文件可再次导入 |
| 未勾选导出 | 未勾选角色时导出按钮置灰或提示请至少选择一条记录 | 未勾选角色导出被拦截 |
| 防重复提交 | 长耗时导入过程中不应重复提交或重复写入 | 导入中重复提交不重复写入 |
| 权限控制 | 导入导出入口仅授权用户可见，接口越权和登录态过期需拦截 | 无导入权限按钮隐藏、无导出权限按钮隐藏、越权导入导出接口被拦截、登录态过期导入导出被拦截 |
| 审计追踪 | 导入导出操作记录操作人、操作时间和导入导出数量 | 导入成功生成审计记录、导出成功生成审计记录 |
