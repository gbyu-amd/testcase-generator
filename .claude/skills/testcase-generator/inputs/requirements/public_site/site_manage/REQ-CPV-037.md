| **类别** | **详细说明** |
| **模块功能概述** | **1、集中管理站点**：本模块用于维护系统中的所有业务站点（如分公司、项目、部门等独立单元），支持站点的增、删、改、查及状态控制。 |
| **页面交互** | **1、入口**：在统一门户页面点击【公共管理】菜单，跳转至公共管理页面，其中包含“站点管理”子页面（或Tab页）。 |
| **需求描述** （新增、编辑、删除、启用、禁用） | **1、新增站点** 1）点击【新增站点】按钮 → 打开新增弹窗。 |
| **异常逻辑说明** | **1、新增/编辑时**： |
| **后置流程处理** | 1. **新增/编辑成功**：关闭弹窗，刷新卡片列表，显示成功提示。 |

**新增站点字段详情表**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **字段名称** | **输入类型** | **必填** | **默认值** | **说明** |
| 站点名称 | 文本输入框 | 是 | 无 | 站点的中文名称，如“华东分公司”、“研发中心”。最大长度100个字符。 |
| 站点编号 | 文本输入框 | 是 | 无 | 唯一标识，如“SITE001”，支持大写字母、数字、下划线，最大长度50个字符。编辑时不可修改。 |
| 状态 | 单选框/下拉框 | 是 | 正常 | 选项：正常、停用。新增时默认为“正常”，停用后站点不可用。 |
| 站点管理员 | 多选组件（下拉树/弹窗选人） | **否** | 无 | 1、从系统用户中选择一个或多个用户作为站点管理员。可支持按用户名、手机号搜索。 2、当一个站点指定一名用户为站点管理员时，用户管理中自动将该用户绑定为该站点；当站点变更管理员时，原用户管理员所属站点不清空，所属的权限跟着业务站点中的来。 |
| 描述 | 文本域（多行文本） | 否 | 空 | 站点的详细描述，长度限制200字符。 |
| 上传站点图片 | 图片上传组件 | 否 | 默认占位图 | 支持格式：jpg, jpeg, png；大小限制：≤2MB；建议尺寸：200x200或16:9缩略图。若未上传，显示系统默认图片。 |

**站点管理 - 站点状态与按钮操作关系**

| **站点状态** | **按钮操作** | **是否可用** | **说明** |
| --- | --- | --- | --- |
| **-** | 新增 | ✅ 可用 | 创建新站点，与当前站点状态无关 |
| **正常** | 编辑 | ✅ 可用 | 可编辑站点信息（默认站点的状态字段不可编辑） |
| 删除 | ✅ 可用 | 需校验无关联数据（用户、部门、角色、业务数据等），公共管理站点不可删除 |
| 停用 | ✅ 可用 | 将站点状态切换为“停用”（公共管理站点无此按钮） |
| **停用** | 编辑 | ✅ 可用 | 可编辑站点信息（包括重新启用时修改状态） |
| 删除 | ✅ 可用 | 需校验无关联数据（停用站点仍可能有关联历史数据），默认站点不可删除 |
| 启用 | ✅ 可用 | 将站点状态切换为“正常” |
| **默认站点（公共管理）** | 编辑 | ✅ 可用 | 可编辑名称、描述、图片等，但状态不可修改（始终正常），站点编号不可改 |
