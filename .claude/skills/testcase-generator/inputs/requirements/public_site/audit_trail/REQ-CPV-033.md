## 原始需求内容

### 系统级审计追踪

**1、系统级审计追踪模块详细功能说明**

|  |  |
| --- | --- |
| **项目** | **描述** |
| **模块功能概述** | 记录系统中所有关键业务操作，满足FDA 21 CFR Part 11等法规对电子记录可追溯性的要求。支持按模块、操作类型、受影响对象等多维度检索，确保所有操作可追溯、可审计。 |
| **需求描述** | **1. 页面布局** • 列表顶部提供多条件搜索：支持按**模块、操作类型、受影响对象**进行精确或模糊搜索。 |
| **页面交互** | • 搜索条件支持多选组合，如选择模块后自动带出该模块下对应的操作类型选项。 |
| **异常逻辑** | • 审计日志存储量过大时，系统应支持按时间范围归档或导出，但不可物理删除。 |

**2、各模块操作类型清单**

|  |  |
| --- | --- |
| **业务模块** | **操作类型** |
| **年度计划设置** | 新增、编辑、删除、复制、提交、撤回、同意、退回、转审、加签、失效、**变更** |
| **任务管理** | 新增、编辑、删除、复制、提交、撤回、**关闭、开始任务**、同意、退回、转审、加签、**升版** |
| **方案模版** | 新增、编辑、删除、复制、提交、升版、失效 |
| **方案编制** | 新增方案、编辑方案、批量导入方案、新增监控项目、编辑监控项目、删除方案、批量删除监控项目、提交方案、撤回方案、同意、退回、转审、加签、失效、升版、导出报告、导入监控项目 |
| **监控项目** | 批量分析、批量导出、数据分析、确认、关闭、**变更** |
| **数据分析** | **暂时不做，用户可用本菜单权限暂时控制** |
| **报告模版** | 新增、编辑、删除、复制、提交、失效、升版 |
| **报告编制** | 新增、编辑、删除、复制、提交、撤回、同意、退回、转审、加签、**升版**、失效、导出报告 |
| **产品配置** | 新增、编辑、删除、导入、导出、批量删除、**导出模版** |
| **用户管理** | 新增、编辑、删除、导入、导出、批量删除、重置密码、失效、重置签名密码 |
| **角色管理** | 新增、编辑、删除、批量删除、分配用户、数据权限 |
| **部门管理** | 新增、编辑、删除 |
| **岗位管理** | 新增、导入、导出、编辑、删除 |
| **工作流配置** | 新增，删除，编辑、配置 |
| **异步任务处理** | 导出文件 |
| **系统服务配置** | 登录配置、用户配置、电子签名密码配置 |
| **个人中心** | 修改头像、修改密码、电子签名密码 |
| **站点管理** | 新增、编辑、批量删除 |

**3、列表字段说明**

|  |  |
| --- | --- |
| **字段名称** | **说明** |
| 序号 | 列表行号，自动生成。 |
| 操作人 | 执行操作的用户账号/姓名。 |
| 操作时间 | 操作发生的具体时间，格式：YYYY-MM-DD HH:MM:SS。 |
| 模块 | 显示对应的操作模块 |
| 操作类型 | 如新增、编辑、提交、同意、退回等，见上述各模块操作类型清单。 |
| 受影响对象 | 操作所影响的业务记录（如计划名称、任务编号、方案名称），支持点击跳转详情页。 |

**说明**：审计日志数据仅供合规审计使用，仅系统管理员可查看全部日志，普通用户仅可查看自己的操作日志。所有日志记录需保留至少6年或根据企业合规要求配置归档策略。
