### 各模块审计追踪详情

通用规则：

每个审计追踪详情页均包含以下两个区域：

![](attachments/IMG-CPV产品规格说明书_0615-20260615-5.png)

**一、基本信息卡片**

|  |  |
| --- | --- |
| **字段** | **说明** |
| 操作模块 | 操作所属的功能模块名称 |
| 操作类型 | 具体的操作类型名称 |
| 操作人 | 执行操作的姓名（同时记录账号） |
| 操作时间 | 格式：年-月-日 时:分:秒 |

**二、内容详情卡片**

根据操作类型不同，展示规则如下：

根据操作类型不同，展示规则如下：

|  |  |
| --- | --- |
| **操作类型** | **展示规则** |
| 新增 | 以属性名/值表格展示该对象的所有字段属性值 |
| 复制 | 以属性名/值表格展示该对象的所有字段属性值（同新增） |
| 删除 | 以属性名/值表格展示被删除对象的所有字段属性值 |
| 编辑 | 以三列表格（字段、旧值、新值）展示有差异的字段，新旧值不一致时颜色标注：旧值使用#FEF0F0背景/#F56C6C文字标注，新值使用#F0F9EB背景/#67C23A文字标注 |
| 批量操作（批量删除、批量分析） | 以列表展示所有数据项的名称和编码 |
| 导入 | 展示导入文件名、导入数量，导入数据列表（名称、代码） |
| 导出 | 导出数据条数，导出数据列表（名称、代码） |
| 审批类操作（提交、撤回、同意、退回、转审、加签） | 展示操作对象的所有属性，并附加电子签名信息卡片 |
| 确认、关闭、变更 | 展示操作对象的相关属性，变更类型展示新旧值对比 |
| 失效 | 展示失效对象信息和失效原因 |
| 升版 | 展示升版对象信息 |
| 配置类操作 | 展示配置项的新旧值对比 |
