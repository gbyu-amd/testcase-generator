## 原始需求内容

### **登录页**

|  |  |
| --- | --- |
| **项目** | **描述** |
| **功能概述** | 提供系统统一登录入口，用户输入全局账号、密码，并选择登录站点（租户），验证通过后跳转至统一门户页面。 |
| **页面布局** | • 顶部：系统Logo + 系统名称“iBatchInsight” |
| **登录流程** | 1. 输入账号密码 2. 根据用户的账号选择站点（站点下拉列表展示用户有权限访问的所有站点，包括“公共管理”及自定义站点） |
| **站点下拉列表** | • 数据来源：根据用户账号查询其有权限访问的站点列表（包含公共管理站点及业务站点） |
| **异常逻辑** | • 账号或密码错误：提示“账号或密码错误” |
| **后置流程处理** | • 记录登录审计日志（操作人、时间、IP、登录站点、登录结果） |
