## 原始需求内容

### 角色权限分配（CPV-RM-01）

**1、角色管理模块详细功能说明**

|  |  |
| --- | --- |
| **项目** | **描述** |
| **模块功能概述** | 维护系统角色信息，支持角色的新增、编辑、删除及状态管理，并为角色分配菜单权限、按钮权限及关联用户，实现基于RBAC模型的权限控制。 |
| **需求描述** | **1. 页面布局** • 顶部提供状态快速切换页签：全部、正常、停用。 • 页面上方为搜索框，支持按**角色名称、角色编号、状态**进行模糊或精确搜索。 • 列表上方提供操作按钮：新增、批量删除、导入、导出。 • 列表默认每页显示50条数据。  **2. 列表属性信息** 复选框、角色编号、角色名称、状态、创建人、创建时间、更新人、更新时间、操作栏（编辑、删除、分配用户）。  **3. 相关按钮操作** • **新增**：点击“新增”按钮，弹出新增角色表单，需填写以下字段： - 角色名称（必填） - 角色编号（必填，唯一标识） - 状态（默认“正常”，可选：正常、停用） - 菜单权限（树形结构展示系统所有菜单，支持勾选；勾选菜单时，自动带出该菜单下的按钮权限） - 按钮权限（展示所选菜单下的按钮权限列表，支持单独勾选） • **编辑**：点击“编辑”按钮，可修改角色信息（角色名称、状态、菜单权限、按钮权限），角色编号不可修改。 • **删除**：点击“删除”按钮，弹出确认提示框：“确定删除该角色吗？”确认后删除。若角色已关联用户，需提示“该角色已关联用户，无法删除”。 • **批量删除**：勾选多个角色后点击“批量删除”，弹出确认框，确认后批量删除。同样需校验是否关联用户。 • **分配用户**：点击“分配用户”按钮，弹出选择用户弹窗，弹窗左侧为组织机构树（取自部门管理），右侧为用户列表，支持按用户名称、手机号搜索。用户列表包含字段：复选框、账号、用户名、邮箱、手机号、状态、创建时间。勾选用户后确认，完成用户与角色的关联。 • **数据权限：**点击“数据权限”按钮，弹出“数据权限”弹窗。 |
| **页面交互** | • 新增/编辑表单中，菜单权限以树形结构展示，支持展开/折叠，支持全选/全不选。 |
| **异常逻辑** | • 新增/编辑时，角色编号已存在则提示“编号已存在，请重新输入”。 • 删除角色时，若角色为系统内置角色（如超级管理员），提示“系统内置角色不可删除”。 • 分配用户时，若用户已拥有该角色，再次分配时应提示“用户已拥有该角色”。 • 选择指定权限，但未给该角色分配任何权限，保存按钮置灰，无法提交。 |
| **后置流程处理** | • 有数据权限的用户角色，在访问：年度计划设置、任务管理、方案模板、方案编制、监控项目、报告模板、报告编制 |

**2、角色列表字段说明**

|  |  |
| --- | --- |
| **字段名称** | **说明** |
| 复选框 | 用于批量操作。 |
| 角色编号 | 角色的唯一标识编码。 |
| 角色名称 | 角色显示名称。 |
| 状态 | 正常 / 停用。停用角色在分配用户时不可选，已关联用户不受影响但角色权限失效。 |
| 创建人 | 创建该角色的用户。 |
| 创建时间 | 创建时间。 |
| 更新人 | 最后修改该角色的用户。 |
| 更新时间 | 最后修改时间。 |
| 操作栏 | 编辑、删除、分配用户。 |

**3、新增/编辑角色字段详情表**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **字段名称** | **输入类型** | **必填** | **默认值** | **说明** |
| 角色名称 | 文本框 | 是 | 无 | 最多50个字符。 |
| 角色编号 | 文本框 | 是 | 无 | 唯一标识，编辑时不可修改。建议格式：ROLE\_XXX。 |
| 状态 | 下拉选择 | 是 | 正常 | 选项：正常、停用。 |
| 菜单权限 | 树形勾选 | 是 | 无 | 展示系统所有菜单，支持按菜单树勾选。 |
| 按钮权限 | 列表勾选 | 否 | 无 | 根据所选菜单动态展示对应的按钮权限列表，支持单独勾选。 |

|  |  |  |
| --- | --- | --- |
| **当前状态** | **操作按钮** | **说明** |
