## 原始需求内容

### **权限管理（公共管理内）**

|  |  |
| --- | --- |
| **项目** | **描述** |
| 模块功能概述 | 1、权限管理：用于管理系统中的权限（即角色或权限实体），支持对权限进行增、删、改、查、配置、停用/启用、分配用户等操作。 2、权限组成：每个权限包含基础信息（权限名称、编号）和配置信息（菜单权限+按钮权限、分配用户）。 3、弹窗拆分：新增/编辑弹窗仅填写基础信息（权限名称、编号）；配置弹窗单独管理功能权限和分配用户（2个Tab页签）。点击权限名称可查看详情（只读模式）。 4、菜单权限范围：公共管理站点下共4个菜单页面：站点管理、用户管理、权限管理、系统服务管理。 5、按钮权限与菜单权限联动：仅展示已勾选菜单下的按钮权限。完整按钮权限矩阵（真实系统数据）：  站点管理：新增、编辑、删除、启用、禁用（5个）  用户管理：新增、编辑、删除、重置密码、失效、导入、导出、批量删除（8个）  权限管理：新增、编辑、删除、启用、停用、配置、批量删除（7个）  系统服务管理：登录配置、用户配置、电子签名密码配置、水印配置（4个）  合计4个菜单、22个按钮权限。 |
| 页面布局 | 1. 搜索区：位于页面上方，包含权限名称（输入框）、权限编号（输入框）。搜索按钮触发列表刷新；重置按钮清空条件并刷新列表。新增按钮位于搜索区右侧。 2. 权限列表： - 首列为复选框。 - 列表字段顺序：复选框、权限名称、权限编号、状态、分配用户、创建人、创建时间、更新人、更新时间、操作。 - 权限名称：蓝色链接，点击打开配置弹窗（只读详情模式，标题显示"权限详情"，隐藏保存按钮）。 - 分配用户：显示"X人"，蓝色链接，点击打开分配用户弹窗。 - 操作栏包含：编辑（打开编辑弹窗）、配置（打开配置弹窗可编辑）、停用/启用、删除。 3. 新增/编辑弹窗（dlg-pub-perm-edit）：520px宽，仅包含基础信息字段。  - 权限名称（文本输入框，必填）  - 权限编号（文本输入框，必填，编辑时不可修改）  - 底部按钮：取消、保存 4. 配置弹窗（dlg-pub-perm-cfg）：900px宽，2个Tab页签。  Tab1 功能权限：左右分栏结构。  - 左侧导航栏：4个菜单页面（站点管理、用户管理、权限管理、系统服务管理），支持搜索过滤。点击导航项滚动定位到右侧对应区域。  - 右侧内容区：顶部全选/清空，下方按菜单分组展示按钮权限（pill标签样式，可勾选）。  Tab2 分配用户：左侧组织机构树 + 右侧用户列表（含搜索、复选框、移除操作）。  - 底部1个保存按钮，统一保存功能权限和分配用户。 5. 详情模式：点击列表中的权限名称，复用配置弹窗但为只读模式。标题改为"权限详情"，保存按钮隐藏，所有编辑控件禁用。 |
| 需求描述（新增/编辑弹窗 + 配置弹窗） | 1、【新增权限】 1）点击列表上方新增按钮，打开新增/编辑弹窗（标题显示"新增权限"）。 2）填写权限名称、权限编号（均为必填）。 3）点击保存，后端校验权限编号唯一性，成功后关闭弹窗并刷新列表。 2、【编辑权限】 1）点击操作栏中的编辑，打开新增/编辑弹窗（标题显示"编辑权限"），回显权限名称和编号。 2）权限编号不可修改（灰色只读），权限名称可改。 3）点击保存，成功后关闭弹窗并刷新列表。 3、【配置权限】 1）点击操作栏中的配置，打开配置弹窗。顶部文本展示当前权限名称和编号（不可编辑）。 2）Tab1 功能权限：左侧菜单导航（4个页面），右侧按钮权限矩阵。支持全选/清空。 3）Tab2 分配用户：左侧组织机构树选择部门，右侧用户列表（含搜索、勾选、移除）。 4）点击保存，统一保存功能权限和分配用户关系，成功后关闭弹窗。 4、【查看详情】 1）点击列表中的权限名称（蓝色链接），打开配置弹窗的只读模式。 2）标题显示"权限详情"，保存按钮隐藏，所有编辑控件（checkbox、按钮等）禁用。 3）仅用于查看，不可修改。点击取消关闭弹窗。 5、【删除权限】 1）点击操作栏中的删除，弹出二次确认框。 2）确认后后端检查是否有关联用户，有则提示并确认后解除关联。 3）删除成功后刷新列表。 6、【停用/启用】 1）正常状态的权限，操作栏显示"停用"按钮，点击后二次确认，确认后状态变更为停用。 2）停用状态的权限，操作栏显示"启用"按钮，点击后状态恢复为正常。 3）停用后拥有该权限的用户不再享有其菜单/按钮权限，但不解除分配关系。 7、【菜单权限与按钮权限联动规则】 - 按钮权限按菜单分组展示，每个菜单对应具体的操作按钮。 - 联动规则：取消某个菜单勾选时，该菜单下按钮权限自动取消；新增勾选菜单时，自动展示其按钮权限（默认不勾选）。 - 公共管理站点完整按钮权限矩阵：  站点管理：新增、编辑、删除、启用、禁用  用户管理：新增、编辑、删除、重置密码、失效、导入、导出、批量删除  权限管理：新增、编辑、删除、分配用户、批量删除  系统服务管理：登录配置、用户配置、电子签名密码配置、水印配置 |
| 异常逻辑说明 | 1、新增/编辑弹窗： - 权限名称为空，提示"请输入权限名称"。 - 权限编号为空，提示"请输入权限编号"。 - 权限编号已存在，提示"编号已存在，请重新输入"。 2、配置弹窗： - 未勾选任何菜单权限，保存时提示"菜单权限不能为空"。 3、删除： - 有关联用户时二次提示"该权限已分配给X个用户，是否仍要删除？"。 4、列表搜索： - 无匹配数据时列表显示"暂无数据"，不清空表头。 5、权限状态停用时，该权限不生效（用户即使被分配，也无法使用其菜单/按钮权限），但不解除分配关系。 |
| 后置流程处理 | 1、新增/编辑成功：关闭弹窗，刷新列表（保持当前搜索条件），显示成功提示。 2、配置保存成功：关闭配置弹窗，刷新列表，显示成功提示。 3、删除成功：刷新列表，显示成功提示。 4、停用/启用成功：刷新列表，更新状态标签和操作按钮。 5、所有失败操作：保持弹窗或页面状态，显示具体错误原因，不刷新列表。 6、权限被删除或停用时：相关用户的该权限即时失效，无需额外通知。 |

权限状态与操作矩阵（含配置操作）

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 当前权限状态 | 编辑 | 配置 | 停用/启用/删除 | 说明 |
| 正常 | ✅ 允许 | ✅ 允许 | 停用 ✅ / 删除 ✅ | 所有操作均可用。点击"停用"可将权限状态变更为停用。停用后不影响已分配记录，但权限立即失效。 |
