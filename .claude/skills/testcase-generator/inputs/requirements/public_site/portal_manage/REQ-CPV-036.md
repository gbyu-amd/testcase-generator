## 原始需求内容

### 统一门户管理

|  |  |
| --- | --- |
| **项目** | **描述** |
| **功能概述** | 用户登录公共管理系统后进入统一门户页面，展示欢迎信息、当前时间，以及所有可访问的业务站点入口（公共管理在此展示入口）。页面提供退出登录功能。 |
| **页面布局** | • 顶部导航栏：显示系统名称、当前用户头像/名称、退出登录按钮 |
| **站点入口交互** | • 点击正常状态的站点卡片，直接跳转至该业务站点的首页（无需再次登录） |
| **页面元素** | • 欢迎词：根据时段显示“上午好/下午好/晚上好” |
| **异常逻辑** | • 跳转站点失败：提示“站点访问失败，请稍后重试”，记录错误日志 |
| **后置流程处理** | 更新用户-站点的最后访问时间 审计追踪，暂不考虑 **门户页面超时登录弹窗暂时不做。** |
