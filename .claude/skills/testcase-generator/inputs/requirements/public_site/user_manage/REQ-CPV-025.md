## 原始需求内容

### **用户账号管理（CPV-UM-01）**

**1、用户管理模块详细功能说明**

|  |  |
| --- | --- |
| **项目** | **描述** |
| **模块功能概述** | 维护系统用户账号信息，支持用户的新增、编辑、删除、启用/停用、失效、密码重置、签名密码重置等操作，并与部门、岗位、角色关联，实现组织架构与权限的统一管理。 |
| **需求描述** | **1. 页面布局** • 左侧为组织机构树形结构（取自部门管理模块），右侧为用户列表。 • 点击左侧部门，右侧列表联动显示该部门下的用户成员。 • 顶部提供状态快速切换页签：全部、正常、停用、已失效。 • 页面上方为搜索框，支持按**用户账号、姓名、部门、状态**进行搜索。 • 列表上方提供操作按钮：新增、批量删除、重置密码、失效、重置签名密码。 • 列表默认每页显示50条数据。  **2. 列表属性信息** 复选框、用户编号、姓名、性别、部门、职位、手机号码、状态、创建时间、操作栏（编辑、删除、重置密码、失效、重置签名密码）。  **3. 相关按钮操作** • **新增**：点击“新增”按钮，弹出新增用户表单，需填写以下字段： - 用户编号（必填，唯一标识） - 用户账号（必填，登录账号，唯一） - 姓名（必填） - 性别（可选） - 手机号码（可选，格式校验） - 邮箱（可选，格式校验） - 归属部门（必填，从部门管理模块树形选择） - 状态（默认“正常”，可选：正常、停用） - 岗位（可选，从岗位管理模块弹窗选择，支持按角色编号、角色名称搜索） - 角色（必填，从角色管理模块弹窗选择，支持多选） - 用户密码（必填，遵循系统密码策略） - 备注（可选） • **编辑**：点击“编辑”按钮，可修改用户信息（除用户编号、用户账号外），弹窗内容同新增。 • **删除**：点击“删除”按钮，弹出确认提示框：“确定删除该用户吗？”确认后删除。若用户存在关联数据（如创建的记录），需提示“该用户已有业务数据，无法删除”或转为逻辑删除。 • **批量删除**：勾选多个用户后点击“批量删除”，弹出确认框，确认后批量删除。 • **重置密码**：点击“重置密码”按钮，弹出输入框，输入新密码（遵循系统密码策略），确认后重置用户登录密码。 • **失效**：点击“失效”按钮，弹出失效原因输入框（必填），提交后用户状态变为“已失效”。失效用户无法登录系统。 • **重置签名密码**：点击“重置签名密码”按钮，弹出输入框，输入新签名密码（遵循系统密码策略），确认后重置用户的电子签名密码。 |
| **页面交互** | • 左侧部门树支持展开/折叠，点击部门高亮，右侧列表自动刷新。 • 搜索框与状态页签联动，实时筛选结果。 • 新增/编辑表单使用弹窗或抽屉式页面，表单支持实时校验（如账号唯一性、手机号格式）。 • 岗位、角色选择器弹窗支持搜索和多选。 • 批量操作时，复选框支持全选/取消全选。 |
| **异常逻辑** | • 新增/编辑时，用户账号已存在则提示“账号已存在，请重新输入”。 • 删除用户时，若用户为系统唯一管理员，提示“无法删除系统唯一管理员”。 • 失效操作时，若未填写失效原因，系统拦截并提示“失效原因不能为空”。 • 重置密码时，新密码需符合系统密码复杂度策略（如长度、包含数字/字母/特殊字符等），否则提示并阻止。 |

**2、用户列表字段说明**

|  |  |
| --- | --- |
| **字段名称** | **说明** |
| 复选框 | 用于批量操作。 |
| 用户编号 | 系统自动生成或手动录入的唯一编号。 |
| 姓名 | 用户真实姓名。 |
| 性别 | 男/女/保密。 |
| 部门 | 所属部门名称。 |
| 职位 | 所属岗位名称。 |
| 手机号码 | 联系方式。 |
| 状态 | 正常 / 停用 / 已失效。 |
| 创建时间 | 用户创建时间。 |
| 操作栏 | 编辑、删除、重置密码、失效、重置签名密码。 |

**3、新增/编辑用户字段详情表**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **字段名称** | **输入类型** | **必填** | **默认值** | **说明** |
| 用户编号 | 文本框 | 是 | 无 | 唯一标识，建议格式：EMP-YYYYMMDD-XXXX。 |
| 用户账号 | 文本框 | 是 | 无 | 登录账号，唯一，不可修改。 |
| 姓名 | 文本框 | 是 | 无 | 最多50个字符。 |
| 性别 | 下拉选择 | 否 | 保密 | 选项：男、女、保密。 |
| 手机号码 | 文本框 | 否 | 无 | 格式校验（11位数字）。 |
| 邮箱 | 文本框 | 否 | 无 | 格式校验（邮箱格式）。 |
| 归属部门 | 树形下拉 | 是 | 无 | 从部门管理模块读取，单选。 |
| 状态 | 下拉选择 | 是 | 正常 | 选项：正常、停用。 |
| 岗位 | 弹窗选择 | 否 | 无 | 从岗位管理模块弹窗选择，支持多选。 |
| 角色 | 弹窗选择 | 是 | 无 | 从角色管理模块弹窗选择，支持多选。 |
| 用户密码 | 密码框 | 是 | 无 | 遵循系统密码策略（长度、复杂度）。 |
| 备注 | 文本域 | 否 | 无 | 补充说明，最多200字符。 |

**用户状态操作矩阵**

|  |  |  |
| --- | --- | --- |
| **当前状态** | **操作按钮** | **说明** |
| **正常** | 编辑、删除、重置密码、失效、重置签名密码 | 用户可正常登录使用。 |
| **停用** | 编辑、删除、重置密码、失效、重置签名密码 | 用户被停用，无法登录，但数据保留。 |
| **已失效** | **原逻辑：编辑、重置签名密码** **现处理为：无任何操作** | **失效用户无法登录，不可删除。无任何操作** |

**说明**：所有操作按钮受权限控制，仅拥有相应权限的用户可见。删除、失效操作需二次确认，并记录审计日志。
