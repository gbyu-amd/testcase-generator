## 原始需求内容

### 用户管理（公共管理内）

|  |  |
| --- | --- |
| **项目** | **描述** |
| **模块功能概述** | 在公共管理系统中统一维护所有用户账号（全局唯一）。用户可被授权访问多个业务站点，每个站点下可独立配置角色和部门（通过后续的权限分配实现）。用户信息创建后自动同步至其关联的业务站点。 |
| **页面布局** | 1. **左侧站点树**：点击任意站点，右侧列表立即刷新并显示该站点用户，同时清空搜索条件。 |
| **需求描述** | **1、新增用户：弹窗表单（Tab页结构）**： |
| **异常逻辑** | **1、文件格式/表头错误**：上传非.xlsx/.xls文件，或表头与模版不匹配（表头的文字内容和模版内容要一模一样）→ 提示“文件格式或表头不正确，请使用标准模版”，不进行逐行校验。 |
| **后置流程处理** | **1、导入成功**：刷新用户列表，弹出成功条数提示，关闭导入弹窗。 |

**新增用户字段详情表**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **字段名称** | **输入类型** | **必填** | **默认值** | **说明** |
| 用户编号 | 文本输入框 | 是 | 无 | 唯一标识，不可重复。支持字母、数字、下划线，最大长度50个字符。 |
| 用户账号 | 文本输入框 | 是 | 无 | 登录账号，唯一，不可重复。支持字母、数字、下划线，最大50个字符； |
| 姓名 | 文本输入框 | 是 | 无 | 真实姓名，最大长度50个字符。 |
| 性别 | 单选框/下拉框 | 是 | 男 | 选项：男、女。 |
| 手机号码 | 文本输入框 | 否 | 无 | 11位数字，大陆手机号格式校验（可扩展国际）。 |
| 邮箱 | 文本输入框 | 否 | 空 | 符合邮箱格式（如xxx@xx.com）。 |
| 用户密码 | 文本输入框（密码框） | 是 | 无 | 新增时必填；编辑时可不填（留空则代表不修改密码）。复杂度建议：至少8位，含字母+数字。 |
| 状态 | 单选框/下拉框 | 是 | 正常 | 选项：正常、停用。停用后该用户不可登录系统。 |
| 备注 | 文本域（多行文本） | 否 | 空 | 选填，长度限制200字符。 |
| 站点授权 | 自定义组件（列表+操作按钮） | 是 | 无 | 1. 包含【新增站点】按钮，点击弹出站点选择弹窗（多选）。 |

**用户状态与操作矩阵**

|  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- |
| **当前用户状态** | **编辑** | **删除** | **重置密码** | **失效** | **说明** |
| **正常** | ✅允许 | ✅允许 | ✅允许 | ✅允许 | 所有操作均可用。失效后状态变为“失效”。 |
| **停用** | ✅允许 | ✅允许 | ✅允许 | ✅允许 | 可编辑重新启用（改状态为正常）；也可直接失效。 |
| **失效** | ❌禁止（或仅允许查看） | ❌不允许 | ❌禁止 | ❌禁止（按钮隐藏） | 失效作为最终状态，不可再重置密码或再次失效。编辑可能仅允许修改备注等无关字段（由业务定）。 |
