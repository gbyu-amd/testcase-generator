## 原始需求内容

### 个人信息管理

**1、个人中心模块详细功能说明**

|  |  |
| --- | --- |
| **项目** | **描述** |
| **模块功能概述** | 为系统用户提供个人信息维护入口，支持查看基本信息、修改个人头像、修改登录密码及电子签名密码，所有修改遵循系统全局安全策略。 |
| **需求描述** | **1. 基本信息展示与维护** 展示当前用户的基本信息，部分字段只读不可修改，支持修改个人头像。 • **用户编号**：系统自动带出，不可更改 • **姓名**：系统自动带出，不可更改 • **手机号码**：系统自动带出，不可更改 • **电子邮箱**：系统自动带出，不可更改 • **部门**：系统自动带出，不可更改 • **职位**：系统自动带出，不可更改 • **修改头像**： a. 默认头像：系统默认生成头像（如姓名首字母或默认图标）。 b. 点击基本信息区域中当前头像旁的"修改头像"按钮，弹出头像上传弹窗。  c. 在弹窗中点击"选择图片"，从本地选择 PNG、JPG、JPEG 格式图片（大小不超过 2MB）。  d. 选择后进入裁剪界面，支持拖拽调整裁剪区域和缩放。  e. 裁剪后实时预览效果，确认后点击"保存"上传至服务器并更新头像显示。  f. 保存后同步更新右上角导航栏头像和个人中心页面头像。  g. 取消则关闭弹窗，不做任何修改。 |
| **操作交互** | • 页面入口：用户点击页面右上角头像/用户名，在下拉菜单中选择"个人中心"进入该页面。 • 基本信息区域采用只读样式，头像区域显示当前头像预览。 |
| **异常逻辑** | • 头像上传：文件格式不符时提示“仅支持PNG、JPG、JPEG 格式”；文件超过2M时提示“文件大小不能超过2M”。 **修改登录密码相关报错文案：** • 当前密码错误：当前密码错误，请重新输入 • 密码强度不足：密码必须包含大写字母、小写字母、数字、特殊字符中的至少X种 • 密码长度不足：密码长度需在X-X位之间 • 密码与历史重复：新密码不能与上次密码相同 **修改电子签名密码相关报错文案：** • 当前电子密码错误：当前电子签名密码错误，请重新输入 • 密码强度不足：电子签名密码必须包含大写字母、小写字母、数字、特殊字符中的至少X种 • 密码长度不足：电子签名密码长度需在X-X位之间 • 密码与历史重复：新签名密码不能与最近X次使用过的密码相同 • 两次密码不一致：两次输入的电子签名密码不一致 |
| **后置流程处理** | • 系统中所有涉及用户展示或选择的位置，均应在用户姓名旁显示用户头像，显示为：[圆形头像] + 姓名（用户账号），涉及范围： 【全局】 a. 全局顶栏：右上角头像按钮处显示圆形头像 + 姓名（用户账号）。 【首页】 a. 待办事项：提交人列显示头像 + 姓名（用户账号）。 b. 消息通知：各通知项左侧显示圆形头像。 【计划管理】 a. 年度计划列表：创建人列显示头像 + 姓名（用户账号）。 b. 年度计划 > 新增/编辑计划弹窗：提醒对象人员选择器中，各人员选项前显示圆形头像。 c. 任务管理 > 新增/编辑任务弹窗：提醒对象人员选择器中，各人员选项前显示圆形头像。 【方案管理】 a. 方案模板列表：创建人列、更新人列显示头像 + 姓名（用户账号）。 b. 方案编制列表：创建者列、更新人列显示头像 + 姓名（用户账号）。 【报告管理】 a. 报告模板列表：创建人列、更新人列显示头像 + 姓名（用户账号）。 b. 报告编制列表：创建人列显示头像 + 姓名（用户账号）。 【配置管理】 a. 产品管理列表：创建人列、更新人列显示头像 + 姓名（用户账号）。 b. 角色管理列表：创建人列、更新人列显示头像 + 姓名（用户账号）。 c. 角色管理 > 分配用户弹窗：用户列表中各用户显示头像 + 姓名（用户账号）。 d. 部门管理列表：负责人列、创建人列、更新人列显示头像 + 姓名（用户账号）。 e. 岗位管理列表：创建人列、更新人列显示头像 + 姓名（用户账号）。 f. 工作流配置卡片：创建人、更新人显示头像 + 姓名（用户账号）。 g. 工作流配置 > 流程设计器：审批人节点显示头像 + 姓名（用户账号）。 h. 异步任务处理列表：创建人列显示头像 + 姓名（用户账号）。 i. 审计追踪列表：操作人列显示圆形头像 + 用户账号。 j. 用户登录日志：用户名列显示姓名（用户账号）。 【个人中心】 a. 基本信息区：顶部显示圆形头像 + "更换头像"按钮。 |

| **字段名称** | **显示方式** | **说明** |

**4、修改电子签名密码字段详情表**

|  |  |  |  |
| --- | --- | --- | --- |
| **字段名称** | **输入类型** | **必填** | **说明** |
| 当前电子密码 | 密码框 | 是 | 验证用户当前电子签名密码。 |
