## 原始需求内容

### 岗位信息维护（CPV-POS-01）

**1、岗位管理模块详细功能说明**

|  |  |
| --- | --- |
| **项目** | **描述** |
| **模块功能概述** | 维护企业组织架构中的岗位信息，支持岗位的新增、编辑、删除、导入导出及状态管理（正常/停用）。岗位信息可用于用户管理及审批流程中的人员筛选。 |
| **需求描述** | **1. 页面布局** • 顶部提供状态快速切换页签：全部、正常、已停用。 • 页面上方为搜索框，支持按**岗位名称**和**岗位状态**进行模糊或精确搜索。 • 列表上方提供操作按钮：新增、导出、导入。 • 列表默认每页显示50条数据。  **2. 列表属性信息** 复选框、岗位名称、岗位编码、岗位排序、状态、创建人、创建时间、更新人、更新时间、操作栏（编辑、删除）。  **3. 相关按钮操作** • **新增**：点击“新增”按钮，弹出新增岗位表单，需填写以下字段： - 岗位名称（必填） - 岗位编码（必填，需校验唯一性） - 岗位排序（必填，数字类型，用于控制显示顺序） - 状态（默认“正常”，可选范围：正常、停用） - 备注（非必填） • **导出**：支持勾选单个或多个岗位，点击“导出”将选中岗位信息导出为Excel文件（.xlsx格式）。若不勾选，默认导出当前列表所有数据（受搜索条件影响）。 • **导入**：点击“导入”弹出导入弹窗，流程如下： - 先下载导入模板（Excel文件，表头包含：岗位名称、岗位编码、岗位排序、状态、备注）。 - 用户按模板填写数据后上传，文件类型仅支持.xlsx和.xls格式，文件选择器默认筛选此类文件。 - 系统校验数据格式及编码唯一性，导入完成后提示成功/失败结果。 • **编辑**：点击操作栏“编辑”按钮，可修改岗位信息（岗位名称、编码、排序、状态、备注）。岗位编码不可修改。 • **删除**：点击“删除”按钮，弹出确认提示框：“确定删除该岗位吗？”确认后删除。若岗位已被用户关联，需提示“该岗位已关联用户，无法删除”。 |
| **页面交互** | • 状态切换页签与搜索框联动，筛选结果实时刷新。 • 新增/编辑表单使用弹窗，支持表单验证（如必填、编码唯一性）。 • 导入时提供模板下载链接，上传后显示导入进度（异步任务）。 |
| **异常逻辑** | • 新增/编辑时，岗位编码已存在则提示“编码已存在，请重新输入”。 • 导入时若数据格式错误（如排序不是数字），在导入结果中提示具体行数和错误原因。 • 删除岗位前校验是否被引用，若已引用则阻止删除并给出提示。 |

**上传文件需要对文件文件格式进行校验：**

|  |  |  |
| --- | --- | --- |
| 序号 | 异常情况 | 对应提示 |
| 2 | Excel表格数据中存在合并单元格 | 文件中存在合并单元格，暂不支持。请处理后重新上传。 |
| 3 | 文件中有表头，但无数据，出对应提示 | 文件中无有效数据，请检查后重新上传 |
| 4 | 加密软件异常失败校验 | 比如加密软件等，导入时，提示：Excel解析失败，请检查格式等信息。 |
| 5 | 上传的文件未确定之前被删除或文件损坏 | 文件已不存在或格式错误，请检查文件。 |
| 6 | 表格中有数据，但表头为空 | 表头信息存在空值，暂不支持分析 |
| 7 | 导入一份空文件 | 文件为空，暂不支持导入 |

**2、新增/编辑岗位字段详情表**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **字段名称** | **输入类型** | **必填** | **默认值** | **说明** |
| 岗位名称 | 文本框 | 是 | 无 | 最多50个字符。 |
| 岗位编码 | 文本框 | 是 | 无 | 唯一标识，不可重复；编辑时不可修改。 |
| 岗位排序 | 数字输入框 | 是 | 0 | 用于控制岗位在列表或下拉中的显示顺序，数字越小越靠前。 |
| 状态 | 下拉选择 | 是 | 正常 | 选项：正常、停用。停用岗位在后续选择中不可见。 |
| 备注 | 文本域 | 否 | 无 | 补充说明，最多200字符。 |

**3、状态操作矩阵**

|  |  |  |
| --- | --- | --- |
| **当前状态** | **操作按钮** | **说明** |
| **正常** | 编辑、删除 | 可编辑信息或删除岗位。删除需二次确认。 |
| **停用** | 编辑、删除 | 可编辑信息或删除岗位。停用岗位仍可编辑启用。 |

**说明**：操作按钮受权限控制，仅拥有相应权限的用户可见。停用岗位在用户管理或流程配置的选择列表中应不可见。
