## 原始需求内容

### 方案模版相关功能（CPV-PT-01）

方案模版模块详细功能说明

|  |  |
| --- | --- |
| **项目** | **描述** |
| **模块功能概述** | 用于创建标准化的方案格式，作为方案编制的依据。支持模版文件上传、在线编辑、版本管理及全生命周期控制。 |
| **需求描述** | **1. 页面布局** • 左侧为产品列表（取自产品管理模块），点击产品可筛选该产品下的方案模版。 |
| **页面交互** | • 新增/编辑页面，模版文件上传区域显示上传进度，上传成功后提供在线预览和编辑入口。 • 在线编辑支持编辑采用ONLYOFFICE进行编辑，编辑后自动保存或手动保存。 • 点击列表行可进入详情页，详情页以只读方式展示所有字段及历史记录。 |
| **异常逻辑** | • 上传非Word/PDF文件时，系统提示“仅支持上传Word或PDF格式文件”。（或者非以上文件格式自动置灰） • 模版编号重复时，保存/提交时提示“编号已存在，请重新输入”。 • 升版时，若原模版已被方案引用，需提示“该模版已被方案引用，升版后旧版本将失效，是否继续？”。 |
| **后置流程处理** | • 模版生效后，可在方案编制模块中被选择引用。 • 模版失效后，方案编制中已引用的旧版本模版仍保留历史数据，但新建方案时不可再选择该失效模版。 |

**新增方案模版字段详情表**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **字段名称** | **输入类型** | **必填** | **默认值** | **说明** |
| 模版名称 | 文本框 | 是 | 无 | 最多100个字符。 |
| 模版编号 | 文本框 | 否 | 手动输入 | 支持手动录入，系统自动校验唯一性。建议格式：TMP-YYYYMMDD-XXXX。 |
| 所属产品 | 下拉选择 | 是 | 无 | 从产品管理模块读取，单选。 |
| 模版文件 | 文件上传 | 是 | 无 | 单文件上传，支持格式：.doc, .docx, .pdf。上传后支持在线预览、编辑及实时保存。 仅允许草稿、、升版中状态的方案模版编辑文档。 |
| 版本号 | 只读 | - | V1.0 | 首次创建为V1.0，每次升版自动增加0.1（如V1.1、V2.0）。系统自动维护。 |
| 创建用户 | 只读 | - | 当前用户 | 系统自动记录。 |

**说明**：新增页面只显示需要用户填写的字段（模版名称、编号、所属产品、模版文件），其他字段由系统自动记录，在详情页展示。

**方案模版状态操作矩阵（无相关权限的按钮不可见）**

|  |  |  |
| --- | --- | --- |
| **当前状态** | **操作按钮** | **说明** |
| **草稿** | 编辑、删除、提交、复制 | 提交后状态变为“已生效”。 |
| **已生效** | 升版、失效、复制 | 升版后生成新版本，新版本状态为“升版中”；失效需填写失效原因，状态变为“已失效”。 |
| **升版中** | 编辑、提交、复制 | 编辑新版本草稿，提交后新版本生效，状态变为“已生效”，旧版本自动变为“已失效”。 |
| **已失效** | 复制 | 复制生成新的草稿记录，新记录状态为“草稿”。 |

**说明**：所有操作按钮均受权限控制，仅拥有相应权限的用户可见。“提交”操作直接使模版生效，无需审批流程（如需审批可后续通过工作流配置扩展）。
