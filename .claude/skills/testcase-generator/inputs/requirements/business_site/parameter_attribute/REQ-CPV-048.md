## 原始需求内容

### 参数与属性维护

本模块包含四个独立子页签：**关键物料属性(CMA)**、**关键工艺参数(CPP)**、**关键质量属性(CQA)**、**过程控制(IPC)**。每个子页签均支持新增、批量导入导出、批量删除等操作，并为工序提供关联的基础数据。

#### **关键物料属性 (CMA)**

|  |  |
| --- | --- |
| **名称** | **需求描述** |
| **需求背景** | 物料属性（如原料粒度、水分、密度等）直接影响产品质量和工艺稳定性，需要统一管理CMA的规格范围（LSL/USL）、目标值、供应商等信息。CMA是工序中物料控制的基础，也是CPV监控的重要对象。 |
| **需求说明** | 1. **列表展示**：展示当前产品下的所有CMA，字段：序号、名称、单位、LSL、TARGET、USL、供应商、描述、操作（编辑、删除）。 |
| **页面布局** | 子页签内独立表格，列宽自适应。顶部按钮区：新增CMA、导入、导出、批量删除。 |
| **异常情况处理** | 1. CMA名称在产品内重复 → 提示“名称已存在”。 |
| **后置处理功能** | 1. 刷新CMA列表，更新左侧产品统计卡片中的CMA数量。 |

#### **关键工艺参数 (CPP)**

|  |  |
| --- | --- |
| **名称** | **需求描述** |
| **需求背景** | CPP是直接影响CQA的工艺参数，需要定义其规格范围、5M1E分类、单位等信息，为工艺控制和CPV监控提供基准。同时CPP将关联到工序中，形成参数-操作对应关系。 |
| **需求说明** | 1. **列表展示**：字段：参数名称、5M1E分类、单位、LSL、TARGET、USL、规格范围（自动生成“LSL～USL”）、描述、操作（编辑、删除）。 |
| **页面布局** | 同CMA子页签，表格列：参数名称、5M1E、单位、LSL、TARGET、USL、格范围、描述、操作。 |

#### **关键质量属性 (CQA)**

|  |  |
| --- | --- |
| **名称** | **需求描述** |
| **需求背景** | CQA是最终产品必须满足的质量指标，需定义接受标准和规格范围，供质量判断及CPV监控使用。CQA通常与多个CPP关联，是工艺优化的目标。 |
| **需求说明** | 1. **列表展示**：字段：属性名称、描述、单位、LSL、TARGET、USL、接受范围、操作（编辑、删除）。 |
| **页面布局** | 同前，表格列：属性名称、描述、单位、LSL、TARGET、USL、接受范围、操作。接受标准可合并到描述列或单独列（建议单独列）。 |
| **异常情况处理** | 同CMA/CPP（唯一性、规格逻辑、删除引用校验）。 |
| **后置处理功能** | 1.刷新CQA列表，更新左侧统计卡片CQA数量，更新工序关联下拉，更新工艺矩阵。 2.相关审计追踪，暂不考虑。 |

#### **过程控制 (IPC)**

|  |  |
| --- | --- |
| **名称** | **需求描述** |
| **需求背景** | IPC用于生产过程中的中间控制，需关联到具体工序，并定义控制类型、方法、标准等，为中控检测和放行提供依据。IPC直接影响工序的执行质量。 |
| **需求说明** | 1. **列表展示**：字段：序号、控制项名称、LSL、TARGET、USL、控制类型、控制方法、控制标准、操作（编辑、删除）。 |
| **页面布局** | 子页签内表格，支持按所属类型筛选。表格列宽适当，控制方法/标准可能较长，支持省略号悬停。 |
| **异常情况处理** | 1. 控制项名称在同一产品下唯一，但若重复不允许提交 |
| **后置处理功能** | 1. 刷新IPC列表，更新工序页签中关联IPC的下拉选项。 2. 相关审计追踪，暂不考虑。 |

**以下是导入文件相关的流程整理：**

|  |  |  |  |
| --- | --- | --- | --- |
| 1 | 上传文件 | 用户上传Excel文件到系统。 | 文件上传至服务器待处理。 |
| 2 | 文件格式校验 | 系统检查文件格式（如扩展名、模板结构等）是否符合要求。 | - 若格式错误，直接提示“文件内容不匹配”，流程终止。 |
| 3 | 逐行数据校验 | 对每一行数据进行业务规则校验（如必填项、数据格式等），记录错误行号和错误原因。多个错误原因用分号隔开。 | 生成错误信息列表（包含行号及对应错误原因）。 |
| 4 | 判断是否有错误 | 检查校验过程中是否发现任何错误。 | -若无错误，转步骤5。 |
| 5 | 产品信息导入 | 将产品信息导入系统，导入完成后提示“XXXX个导入成功”。 | 显示成功提示，导入完成。 |
| 6 | 生成错误报告 | 整份文件存在一个以上错误时，不允许导入，并生成TXT格式错误报告。 报错内容格式： | 提供错误报告TXT文件下载，用户可获取详细失败记录。 |
