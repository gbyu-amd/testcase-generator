### **ONLYOFFICE文档编辑权限汇总表**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **模块** | **文档对象** | **状态** | **ONLYOFFICE操作权限** | **说明** |
| **方案模版** | 模版文件 | 草稿 | 可编辑 | 新增或编辑草稿时，支持手动上传后在线编辑、实时保存。 |
|  | 已生效 | 只读 | 模版生效后不可修改，仅可预览。如需变更，需进行“升版”操作。 |
|  | 升版中 | 可编辑 | 对已生效模版点击“升版”后进入此状态，可编辑新版本内容。 |
|  | 已失效 | 只读 | 失效模版仅可预览，不可编辑。 |
| **方案编制** | 方案文档 | 草稿 | 可编辑 | 方案草稿阶段，引用方案模版，用户可在线编辑文档内容。 |
|  | 审批中 | 只读 | 提交审批后文档锁定，不可编辑。可以加批注 |
|  | 已退回 | 可编辑 | 审批退回后，用户可修改文档后重新提交。（附件、签批页功能、不可插入分析结果） |
|  | 已生效 | 只读 | 方案生效后文档固化，仅可预览。如需变更，需进行“升版”。 |
|  | 升版中 | 可编辑 | 升版过程中编辑新版本文档，原版本仍为已生效。 |
|  | 已失效 | 只读 | 失效方案仅可预览，不可编辑。 |
| **报告模版** | 模版文件 | 草稿 | 可编辑 | 新增或编辑草稿时，支持手动上传在线编辑、实时保存。（包括插入附件（包含新增附件，上传附件、插入附件、附件清单）、插入签批页） |
|  | 已生效 | 只读 | 模版生效后不可修改，仅可预览。升版操作可创建新版本。 |
|  | 升版中 | 可编辑 | 对已生效模版升版后进入此状态，可编辑新版本内容。 |
|  | 已失效 | 只读 | 失效模版仅可预览，不可编辑。 |
| **报告编制** | 报告文档 | 草稿 | 可编辑 | 可引用报告模版，报告草稿阶段，用户可在线编辑（包括插入附件（包含新增附件，上传附件、插入附件、附件清单）、插入签批页、插入分析结果等）。 |
|  | 审批中 | 只读 | 提交审批后文档锁定，不可编辑。创建人可“撤回”回到草稿。可以加批注 |
|  | 已退回 | 可编辑 | 审批退回后，用户可修改文档后重新提交。 |
|  | 已撤回 | 可编辑 | 创建人撤回审批后，文档回到草稿状态，可编辑。 |
|  | 已生效 | 只读 | 报告生效后文档固化，仅可预览。如需变更，需进行“升版”。 |
|  | 升版中 | 可编辑 | 升版过程中编辑新版本文档，原版本仍为已生效。 |
|  | 已失效 | 只读 | 失效报告仅可预览，不可编辑。 |

**总结说明**

* **可编辑状态**主要集中在各模块的**草稿、已退回、升版中**（以及报告编制的**已撤回**），这些状态下文档处于“草稿”或“修订”阶段，允许用户修改。
* **只读状态**包括**已生效、已失效、审批中**，文档内容已固化或流转中，不可修改，仅可预览。
* **特殊规则**：报告编制在“审批中”时，创建人可“撤回”进入可编辑状态；其他模块无此操作。
