## 原始需求内容

### 报告编制支持导出

【需求背景】

支持将报告编制中关联的ONLYOFFICE在线文档导出为本地文件。导出功能受权限控制，仅拥有导出权限的用户可见操作入口。

【需求描述】

|  |  |
| --- | --- |
| **项目** | **描述** |
| **需求描述** | **1. 导出入口** • **列表页导出按钮**：在报告编制列表的操作栏中增加“导出”按钮，支持权限配置（需拥有导出权限的用户可见）。 • **ONLYOFFICE预览详情页导出按钮**：在报告编制预览（ONLYOFFICE）中增加“导出”按钮，点击后同样触发导出弹窗。  **2、导出内容：** - 全部导出：导出报告编制正文 + 所有附件  **3. 导出文件命名规则** • 导出文件为ZIP压缩包，命名格式： {报告名称}\_{导出项数量}\_{导出日期}.zip • **示例**：阿莫西林胶囊报告V1.0\_120项\_20231027.zip • 其中“120项” = 报告正文（1项） + 勾选导出的附件数量 • 导出日期格式：YYYYMMDD  **4. 导出内容说明** • 报告正文：按正文文件类型（Word/PDF）导出 • 附件：按原始格式导出，文件名保持原始附件名称 • 所有文件打包至同一ZIP压缩包中，根目录下不创建子文件夹  **5. 权限控制** • 列表页“导出”按钮需绑定权限标识，仅拥有导出权限的用户可见 • 详情页导出按钮同理，需与列表页权限保持一致 |
| **页面交互** | • 导出按钮位于ONLYOFFICE工具栏中，与打印、下载等按钮并列。 • 点击导出按钮后，浏览器自动开始下载文件，无需额外弹窗确认。 • 下载过程中显示加载提示（如“正在导出，请稍候…”）。 • 导出完成后，浏览器自动保存文件至默认下载目录。 |
| **异常逻辑** | • 若ONLYOFFICE文档不存在或无法读取，提示“文档不存在，导出失败”。 • 若导出过程中网络中断，提示“导出失败，请重试”。 • 若用户无导出权限，导出按钮不显示或置灰。 • 若文档内容为空或格式异常，提示“文档内容异常，导出失败”。 |
| **后置流程处理** | • 导出操作需记录审计日志（操作人、操作时间、方案名称、文档名称）。 • 导出文件不占用系统存储空间，直接由前端/后端生成并传输给用户。 |
