## 原始需求内容

### 新增工艺参数配置功能

【业务背景】

在CPV系统中，方案编制时需要引用大量工艺参数（如CQA、CPP等），传统做法是在每个方案编制中重复定义或导入，造成数据冗余和不一致。为统一管理企业级工艺参数库，提高复用性和标准化水平，需在产品管理模块中增加独立的工艺参数配置功能，支持对每个产品维护其关联的工艺参数列表（包括名称、类型、质量标准等），后续方案编制时可直接从该库中选取，避免重复录入，确保数据一致性。

【需求描述】

|  |  |
| --- | --- |
| **项目** | **内容** |
| **需求描述** | **1. 入口与权限** • 在产品管理列表的操作栏中增加“配置工艺参数”按钮。 • 按钮受权限控制，仅授权用户可见。 • 点击按钮弹出“配置工艺参数”弹窗；  **2. 弹窗功能** 弹窗顶部显示产品名称，列表展示该产品下已配置的工艺参数。支持以下操作： • **新增**：点击“新增”按钮，弹出表单，填写： - 工艺参数名称（必填，文本） - 参数类型（下拉，必填、下拉选择，选项内容为：CQA、CMA、CPP、IPC、其他） - 质量标准（非必填，支持文本输入，如“95~105%”或“≤0.1%”） - 标准类型（非必填，下拉，选项内容为：法规限、内控限、警戒限，纠偏限） - 备注（选填，文本域） • **批量删除**：勾选列表中的复选框，点击“批量删除”，二次确认后删除。 • **批量导入**： - 点击“导入”按钮，弹出导入弹窗，先下载Excel模板（表头：工艺参数名称、参数类型、质量标准、标准类型、备注）。 - 用户按模板填写后上传，系统校验数据合法性，成功后批量新增/更新（根据参数名称+产品去重）。 • **导出**：支持将当前产品的工艺参数列表导出为Excel。 • **编辑/删除**：每条记录后提供编辑、删除按钮。  **3. 数据保存** • 新增或编辑后点击“确定”，立即生效，并刷新列表。 • 关闭弹窗前若未保存，提示用户是否保存。  **4. 与方案编制的联动** • 在方案编制页面的配置相关性分析时，可选择该产品已配置的工艺参数； • 导入时自动带出参数名称、类型、质量标准、标准类型等，无需重复填写。 |
| **异常流程处理** | • **重复参数名称**：同一产品下，参数名称不能重复。新增或导入时若重复，提示“参数名称已存在”，并阻止保存。 • **导入格式错误**：上传非Excel文件或表头不匹配，提示“请使用系统模板”。 • **必填项缺失**：新增或编辑时，若未填参数名称、类型、质量标准，高亮字段并提示。 • **批量删除确认**：删除前弹框“确定删除选中的X个参数吗？删除后不可恢复，且已引用的方案将受影响。” • **引用检查**：若某工艺参数已被方案编制引用，则不允许删除，提示“该参数已被方案【XXX】引用，请先解除引用”。 |
| **后置处理流程** | • 保存后更新产品下的工艺参数缓存，确保方案编制模块能获取最新列表。 • 本次不考虑审计日志 • 产品删除时，需级联删除其下的工艺参数。 |

**新增工艺参数字段（可批量导入）**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **字段名称** | **输入类型** | **必填** | **默认值** | **说明** |
| 工艺参数名称 | 文本框 | 是 | 无 | 具体工艺参数的指标名，如“含量均匀度”。 |
| 参数类型 | 下拉选择 | 是 | CQA | 选项：CQA、CPP、CMA、IPC、其他。 |
| 质量标准 | 文本框 | 否 | 无 | 如X≤result≤Y。 |
| 标准类型 | 下拉框 | 否 | 无 | 选项内容为：法规限、内控限、警戒限，纠偏限 |
| 备注 | 文本域 | 否 | 无 | 额外说明。 |

导入监控项目，首先需要先下载模版，**文件类型**仅支持.xlsx和.xls格式的Excel文件，弹窗的文件选择器应默认筛选此类文件，其他文件类型默认不可见。

**下载Excel模版，鼠标悬停表头信息展示为：**

|  |  |  |
| --- | --- | --- |
| Excel模版表头字段信息 | 类型 | 鼠标悬停在表头上文字显示 |
| 工艺参数名称 | 文本框 | 必填，唯一性、最大支持200个字符 |
| 参数类型 | 下拉框，选项内容为：CQA、CMA、CPP、IPC、其他 | 必填、下拉选择，选项内容为：CQA、CMA、CPP、IPC、其他 |
| 质量标准 | 文本输入，输入格式为：XX<B<YY或文本内容，若输入XX<B<YY格式，数据分析时可自动带出 | 非选填，文本输入，输入格式为：X<result<Y或X≤result≤Y或文本内容，若输入X<result<Y格式，数据分析时可自动带出。 |
| 标准类型 | 下拉框 | 非必填、下拉选择，选项内容为：参考线配置模块的参考限，例如法规限、内控限、警戒限、纠偏限等 |
| 备注 | 文本 | 选填 |

**以下是导入文件相关的流程整理：**

|  |  |  |  |
| --- | --- | --- | --- |
| 1 | 上传文件 | 用户上传Excel文件到系统。 | 文件上传至服务器待处理。 |
| 2 | 文件格式校验 | 系统检查文件格式（如扩展名、模板结构等）是否符合要求。 | - 若格式错误，直接提示“文件内容不匹配”，流程终止。 - 若格式正确，继续下一步。 |
| 3 | 逐行数据校验 | 对每一行数据进行业务规则校验（如必填项、数据格式等），记录错误行号和错误原因。多个错误原因用分号隔开。 | 生成错误信息列表（包含行号及对应错误原因）。 |
| 4 | 判断是否有错误 | 检查校验过程中是否发现任何错误。 | -若无错误，转步骤5。 - 若有错误，转步骤6。 |
| 5 | 工艺参数导入 | 将工艺参数导入系统，导入完成后提示“XXXX个工艺参数导入成功”。 | 显示成功提示，导入完成。 |
| 6 | 生成错误报告 | 整份文件存在一个以上错误时，不允许导入，并生成TXT格式错误报告。 报错内容格式： | 提供错误报告TXT文件下载，用户可获取详细失败记录。 |

|  |  |  |

| 7 | 导入一份空文件 | 文件为空，暂不支持导入 |
