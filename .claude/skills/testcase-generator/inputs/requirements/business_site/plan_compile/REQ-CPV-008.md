## 原始需求内容

### 【方案编制】新增相关性分析

**【需求背景】**

目前CPV流程中，方案编制直接定义监控项目，缺乏数据驱动的依据。为提升监控的科学性，需要先通过相关性分析识别关键工艺参数与质量属性之间的关联，再基于分析结果创建监控项目。因此，需在方案编制页面新增“相关性分析”Tab，支持用户配置两种类型的相关性分析，并生成分析记录，为后续监控项目创建提供依据。

**【需求描述】**

|  |  |
| --- | --- |
| **项目** | **内容** |
| **需求描述** | **1. 方案编制三Tab设计 1）详情页**：规则同原有（方案名称、编号、所属产品、CPV任务、方案模版）。 **2）相关性分析Tab**： 点击“添加”按钮，弹出配置弹窗，支持两种统计分析工具： |
| **异常情况说明** | 1. 方案编制新增提交时，若相关性分析列表为空 → 提示“请至少添加一条相关性分析”。 |
| **后置处理功能** | 1. 方案编制新增提交后，系统根据每条相关性分析定义，在“方案执行 > 相关性分析”二级菜单中创建对应的分析记录（状态“未分析”），并关联当前方案。 |

**新增/编辑相关性分析弹窗字段详情表**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **字段名称** | **输入类型** | **必填** | **默认值** | **说明** |
| 统计分析工具 | 单选按钮 | 是 | 相关性分析 | 选项：相关性分析、Pearson相关性分析 |
| 目标变量（当工具=相关性分析时） | 下拉选择 | 是 | 无 | 从该产品已配置的工艺参数库中选择一个（通常为CQA） |
| 预测变量（当工具=相关性分析时） | 多选下拉 | 是 | 无 | 从同一工艺参数库中选择至少两个（通常为CPP） |
| 工艺参数（当工具=Pearson相关性分析时） | 多选下拉 | 是 | 无 | 至少选择2个定量变量（可跨类型） |
| 备注 | 文本域 | 否 | 无 | 补充说明，最多200字符 |
