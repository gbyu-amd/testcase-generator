## 原始需求内容

### 操作

【需求背景】

一个产品一个工序中可能存在多个操作，需要统一管理不同的操作，并实时展示每个操作关联的工序、设备、CMA、CPP、CQA、 IPC。

【需求描述】

|  |  |
| --- | --- |
| **名称** | **需求描述** |
| **需求说明** | 1. **列表展示**：展示当前选中产品下的所有操作，字段包括：流程名称、描述、关联工序、CMA、CPP、CQA、IPC、创建人、创建时间、操作（编辑、删除）。 |
| **页面布局** | 1、表格位于右侧TAB页签“操作”下，表格列：操作名称、描述、CMA、CPP、CQA、IPC、创建人、创建时间、描述、操作（编辑、删除）。 2、表格支持分页（每页30条），可不考虑分页。 |
| **异常情况处理** | 1. 新增/编辑时操作名称在产品内重复 → 提示“该操作名称已存在”。 |
| **后置处理功能** | 1. 新增/编辑/删除操作后，刷新操作列表。 |

注：所有多选下拉均需支持搜索、全选、清空功能。选择后，在列表中应显示已选中的数量或名称摘要。

#### 操作支持导入导出

【需求背景】

在产品工艺管理模块中，“操作”页签用于维护生产过程中的各类操作单元（如加料、混合、干燥等）。当前用户需逐个手动录入或编辑操作信息，效率较低，尤其在批量初始化或工艺变更时容易出错。因此需增加操作的导入导出功能，支持用户通过Excel模板批量导入、导出操作数据，提升数据维护效率，便于跨系统数据交换和备份。

【需求描述】

|  |  |
| --- | --- |
| **名称** | **需求描述** |
| **需求描述** | **一、导出功能** 1. **操作入口**：操作列表页顶部操作按钮区增加“导出”按钮。用户可勾选操作列表中的复选框（支持多选）后点击导出，若未勾选则导出当前列表所有操作（受搜索筛选影响）。 |
| **页面布局** | 操作页签（操作列表页）顶部操作栏新增“导入”、“导出”按钮，位于“新增”按钮右侧。列表支持勾选复选框（多选）。分页每页20条。导入弹窗为模态框，居中显示，宽度500px。 |
| **交互说明** | - 导出时，若未勾选任何操作，点击导出按钮后导出当前列表全部操作（含搜索筛选结果）。 |
| **异常逻辑** | 1. 文件格式非.xlsx/.xls → 提示“请上传Excel文件”。 |
| **后置处理功能** | 1. 导出操作记录审计日志（操作人、时间、导出文件名、导出数量）。 |
| **导出审计追踪** | **1、审计追踪搜索：**增加操作**导入操作、导出操作**的操作类型 **2、审计追踪列表：** 导出：受影响对象（{产品名称}\_操作\_{导出数量}\_{时间戳}.xlsx） 导入：受影响对象（{产品名称}\_操作\_{导出数量}\_{时间戳}.xlsx） **3、受影响对象：** 1）基本信息：操作人、操作时间、操作模块（产品工艺管理）、操作类型（导出操作） 2）内容详情：导出环境、批量导出XX个操作、成功XXX条、失败XXX条、导出文件名称《XXXXXXX》，文件；类型xlsx 3）下方展示操作列表：下方展示操作列表：操作名称、描述、关键工序、CMA、CPP、IPC、CQA |
| **导入审计追踪** | 基本信息：操作人、操作时间、操作模块（产品工艺管理）、操作类型（导入操作） 内容详情：导入文件名、导入数量、导入详情 下方展示工序列表：下方展示工序列表：操作名称、描述、关键工序、CMA、CPP、IPC、CQA |
