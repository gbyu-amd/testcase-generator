### 关键设备

【需求背景】

关键设备是影响工艺稳定的重要资产，需要为每个工序关联设备，并跟踪设备的实时状态，为设备运维提供信息。

【需求描述】

|  |  |
| --- | --- |
| **名称** | **需求描述** |
| **需求背景** | 关键设备是影响工艺稳定性和产品质量的重要因素。在实际生产中，同一设备可能用于多个操作，设备状态（正常运行、维护中等）直接影响生产安排。需要为每个产品建立关键设备台账，与工序关联，并支持状态跟踪，为设备管理、维护计划以及CPV监控提供基础数据。目前缺乏统一的关键设备管理，导致设备信息分散，无法追溯设备对工艺参数的影响。 |
| **需求说明** | 1.**列表展示**：展示当前选中产品下的所有关键设备列表，字段包括：序号、设备名称、状态、操作（编辑、删除）。 |
| **页面布局** | 1.表格位于右侧TAB页签“关键设备”下，表格列：序号、设备名称、状态、操作（编辑、删除）。 2.顶部有“新增”按钮。表格支持分页（每页20条）。 3.设备名称过长时显示省略号，鼠标悬停显示全称。 4.状态用不同颜色的标签展示（绿色=正常运行，橙色=维护中，红色=维修中，灰色=待机/停用）。 |
| **异常情况处理** | 1. 新增/编辑时设备名称在当前产品下重复 → 若用户尝试添加完全相同的设备名称，则提示“该设备已存在，请勿重复添加”。 |
| **后置处理功能** | 1. 新增/编辑/删除成功后，刷新设备列表。 |

**新增/编辑关键设备字段表**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **字段名称** | **输入类型** | **是否必填** | **默认值** | **说明** |
| 设备名称 | 文本框 | 是 | 无 | 最多100字符，在产品内唯一 |
| 状态 | 下拉单选 | 是 | 正常运行 | 选项：正常运行、维护中、维修中、待机、停用 |

注：编辑时设备名称若修改，需重新校验唯一性（排除自身当前记录）。

#### 关键设备支持导入导出

【需求背景】

在产品工艺管理模块的“关键设备”页签中，用户需要批量维护关键设备信息（如设备名称、状态等）。当前仅支持单条录入，效率较低，尤其在初始化或批量调整时容易出错。因此需增加关键设备的导入导出功能，支持用户通过Excel模板批量导入、导出设备数据，提升数据维护效率和准确性，并满足数据迁移、备份及跨系统交换需求。

|  |  |
| --- | --- |
| **名称** | **需求描述** |
| **需求背景** | 在产品工艺管理模块的“关键设备”页签中，用户需要批量维护关键设备信息（如设备名称、状态等）。当前仅支持单条录入，效率较低，尤其在初始化或批量调整时容易出错。因此需增加关键设备的导入导出功能，支持用户通过Excel模板批量导入、导出设备数据，提升数据维护效率和准确性，并满足数据迁移、备份及跨系统交换需求。 |
| **需求描述** | **一、导出功能** 1. **操作入口**：关键设备列表页顶部操作按钮区增加“导出”按钮。用户可勾选设备列表中的复选框（支持多选）后点击导出，若未勾选则导出当前列表所有设备（受搜索筛选影响）。 |
| **页面布局** | 关键设备页签（设备列表页）顶部操作栏新增“导入”、“导出”按钮，位于“新增”按钮右侧。列表支持勾选复选框（多选）。分页每页20条。导入弹窗为模态框，居中显示，宽度500px。 |
| **交互说明** | - 导出时，若未勾选任何设备，点击导出按钮后导出当前列表全部设备（含搜索筛选结果）。 |
| **异常逻辑** | 1. 文件格式非.xlsx/.xls → 提示“请上传Excel文件”。 |
| **后置处理功能** | 1. 导出操作记录审计日志（操作人、时间、导出文件名、导出数量）。 |
| **导出审计追踪** | **1、审计追踪搜索：**增加操作导入关键操作、导出关键操作的操作类型； **2、审计追踪列表：** 导出：受影响对象（{产品名称}\_关键操作\_{导出数量}\_{时间戳}.xlsx） 导入：受影响对象（{产品名称}\_关键操作\_{导出数量}\_{时间戳}.xlsx） **3、受影响对象：** 1）基本信息：操作人、操作时间、操作模块（产品工艺管理）、操作类型（导出关键设备） 2）内容详情：导出环境、批量导出XX 个操作、成功XXX条、失败XXX条、导出文件名称《XXXXXXX》，文件；类型xlsx 3）下方展示操作列表：下方展示工序列表：设备名称、关键操作、状态 |
| **导入审计追踪** | 基本信息：操作人、操作时间、操作模块（产品工序管理-工序）、操作类型（导入关键设备） 内容详情：导入文件名、导入数量、导入详情 下方展示工序列表：下方展示工序列表：设备名称、关键操作、状态 |
