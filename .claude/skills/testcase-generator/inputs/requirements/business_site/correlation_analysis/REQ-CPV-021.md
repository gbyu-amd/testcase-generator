## 原始需求内容

### 支持插入相关性分析模块产生的分析结果

【需求背景】

在报告编制过程中，用户需要将已执行的相关性分析结果或监控项目分析结果插入到报告中，作为结论依据或数据展示。当前插入方式较为单一，无法灵活选择分析结果中的特定内容（如是否包含结论、数据表等），且缺少对插入数量的限制和图表展示样式的规范。因此需优化插入分析结果弹窗，支持多源选择、精细控制插入内容，并统一图表展示标准。

【需求描述】

|  |  |
| --- | --- |
| **项目** | **内容** |
| **需求描述** | **1. 插入分析结果入口** 在报告编制页（ONLYOFFICE编辑器）的工具栏中，提供“插入分析结果”按钮，点击后弹出模态框。  **2. 弹窗结构** 弹窗包含两个页签：**相关性分析**、**监控项目**。  **1）相关性分析页签** • 左侧：产品列表，选择产品后右侧联动显示该产品下的方案下拉框（支持按方案筛选）。 • 右侧上方：搜索框，支持按工艺参数名称、工艺参数编号、方案名称搜索。 • 右侧下方：列表，字段包括： - 全选复选框 - 编号、工艺参数名称、分析状态、参数类型、统计分析工具、分析结果名称、方案名称 - **分析结果**：开关，默认打开（即插入分析图时会包含结论文字） - **数据表数据**：开关，默认只选中第一行（即只插入第一个数据表，用户可手动勾选多行）  **2）监控项目页签** • 左侧：产品列表，选择产品后联动显示方案。 • 右侧上方：搜索框，支持按监控项目名称、监控项目编号、方案名称搜索。 • 右侧下方：列表，字段包括： - 全选复选框 - 编号、工艺参数名称、分析状态、监控类型、分析结果名称、质量标准、标准类型、方案名称 - **分析结果**：开关，默认打开 - **数据表数据**：开关，默认只选中第一行  **3. 插入数量限制** • 单次批量插入分析结果的数量上限为 **100个**（两个页签的分析结果数量加在一起）。 • 超过限制时系统拦截提示：“单次最多支持插入100个分析结果，当前已选择X个，请分批插入。”  **4. 分析图展示样式要求** • 分析图需增加**外围框线**（边框），与报告正文内容形成视觉区分。 • 分析图宽度需**占满A4纸的有效宽度**（约16-18cm，或按页面边距自适应）。 • 图表内**数据点必须清晰展示**，不可出现压缩、拉伸、模糊等影响数据判读的情况。 |
| **异常情况说明** | 1. 未选择任何分析结果时，置灰“确定”按钮 或 提示“请至少选择一个分析结果”。 2. 插入数量超过100个 → 拦截并提示具体数量，不予提交。 5. 插入操作因网络超时失败 → 提示“插入失败，请检查网络后重试”。 |
| **后置处理功能** | 1. 插入成功后，弹窗自动关闭，报告中即时显示所选的图表和结论。 2. 插入的分析结果支持在报告内调整位置和大小。 3. 若用户选择包含“数据表数据”，则报告中以表格形式呈现所选的数据行。 |
