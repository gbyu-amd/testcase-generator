## 原始需求内容

#### 工序支持导入导出

【需求背景】

在产品工艺管理模块的“工序”页签中，用户需要批量维护工序信息，如新增、编辑、删除等操作。当前仅支持单条录入，效率较低，尤其在初始化或批量调整时容易出错。因此需增加工序的导入导出功能，支持用户通过Excel模板批量导入、导出工序数据，提升数据维护效率和准确性，并满足数据迁移、备份及跨系统交换需求。

【需求描述】

|  |  |
| --- | --- |
| **名称** | **需求描述** |
| **需求描述** | **一、导出功能** 1. **操作入口**：工序列表页顶部操作按钮区增加“导出”按钮。用户可勾选工序列表中的复选框（支持多选）后点击导出，若未勾选则导出当前列表所有工序（受搜索筛选影响）。 |
| **页面布局** | 工序页签（工艺步骤列表页）顶部操作栏新增“导入”、“导出”按钮，位于“新增”按钮右侧。列表支持勾选复选框（多选）。分页每页50条。导入弹窗为模态框，居中显示，宽度500px。 |
| **交互说明** | - 导出时，若未勾选任何工序，点击导出按钮后导出当前列表全部工序（含搜索筛选结果）。 |
| **异常逻辑** | 1. 文件格式非.xlsx/.xls → 提示“请上传Excel文件”。 |
| **后置处理功能** | 1. 导出操作记录审计日志（操作人、时间、导出文件名、导出数量）。 |
| **导出审计追踪** | **1、审计追踪搜索：**增加导入工序、导出工序的操作类型 **2、审计追踪列表：** 导出：受影响对象（{产品名称}\_工序\_{导出数量}\_{时间戳}.xlsx） 导入：受影响对象（{产品名称}\_工序\_{导出数量}\_{时间戳}.xlsx） **3、受影响对象：** 1）基本信息：操作人、操作时间、操作模块（产品工艺管理）、操作类型 2）内容详情：导出环境、批量导出XX个操作、成功XXX条、失败XXX条、导出文件名称《XXXXXXX》，文件；类型xlsx  2）下方展示工序列表：工序名称、标准时长、关键操作、描述 |
| **导入审计追踪** | 基本信息：操作人、操作时间、操作模块（产品工艺管理）、操作类型（导入工序） 内容详情：导入文件名、导入数量、导入详情 下方展示工序列表：工序名称、标准时长、关键操作、描述 |
