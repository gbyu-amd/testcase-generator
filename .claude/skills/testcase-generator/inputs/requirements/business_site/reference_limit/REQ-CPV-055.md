## 原始需求内容

### 参考限配置

【需求背景】

在CPV系统的数据分析模块中，用户需要在控制图、过程能力图等图表中叠加不同类型的参考线（如法规限、内控限、警戒限、纠偏限），用于直观判断数据是否超出预设阈值。目前参考线缺乏统一管理，用户每次分析时需手动输入，效率低且易出错。因此需在“配置管理”下新增“质量规则配置”二级菜单，其中“参考线配置”页签用于集中管理所有参考线的定义，包括默认参考线及用户自定义参考线，并支持在分析图表中自动应用。

【需求描述】

|  |  |
| --- | --- |
| **名称** | **需求描述** |
| **需求描述** | **一、菜单入口**：在系统左侧导航栏“配置管理”一级菜单下，新增“质量规则配置”二级菜单。点击后进入页面，默认选中“参考线配置”页签。 |
| **页面布局** | 页面分为上下两部分： 顶部搜索区（参考线名称、编号输入框，搜索按钮）； 中部操作按钮区（新增按钮）； 下部参考线列表（表格）。 表格列包括：移动（拖拽手柄）、序号、参考限名称、编号、颜色（色块预览）、线条、粗细、描述、创建人、创建时间、更新人、更新时间、操作（编辑、删除）。 列表支持分页（每页20条）。 默认参考线的删除按钮置灰不可点。 拖拽排序时，鼠标显示移动图标。 |
| **交互说明** | - 新增/编辑参考线使用模态框，表单字段见下表。其中颜色、线条、粗细三个选项采用**平铺按钮组**形式展示（便于用户直观选择），而非下拉列表。 |
| **异常逻辑** | 1. 新增/编辑时参考线名称或编号与已有记录重复 → 提示“名称/编号已存在”。 |
| **后置处理功能** | 1. 新增/编辑/删除/排序操作先不记录审计日志。 2.配置的参考线仅在下述分析图类型中生效，其他图表不显示该参考线： - 单值移动极差控制图 - 单值控制图 - 百分位数控制图 - Xbar-S控制图 - Xbar-R控制图 - 持续监控图 - 趋势图 - 单折线图 - 多折线图 - P控制图 |
| **数据兼容性** | 老数据由于系统只有默认的4条参考线，老数据重新点击开始分析时，按照最新的参考限配置来 |

**新增/编辑参考线字段表（平铺样式）**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **字段名称** | **输入类型** | **是否必填** | **默认值** | **说明** |
| 参考限名称 | 文本框 | 是 | 无 | 最多50字符，全局唯一 |
| 编号 | 文本框 | 是 | 无 | 全局唯一，建议英文大写+下划线，如“REG\_LIMIT” |
| 颜色 | 平铺色块组 + 文本输入框 | 是 | #000000 | 18种常见色块平铺展示，用户点击色块即选中；同时提供十六进制颜色码输入框，两者联动。色块网格布局，每行6个。 |
| 线条 | 平铺按钮组 | 是 | 实线 | 两个按钮并排：“实线”、“虚线”。选中后高亮。 |
| 粗细 | 平铺按钮组 | 是 | 1px | 三个按钮并排：“1px”、“2px”、“3px”。选中后高亮。 |
| 描述 | 文本域 | 否 | 无 | 最多200字符 |
| 预览 | 预览区域 | — | — | 实时展示当前所选颜色、线条、粗细组合的示例线（如一段横线），用户确认样式。 |

注：编辑时，默认参考线的名称、编号也可修改，但需保持唯一性；默认参考线的删除按钮始终置灰。

**默认参考线初始数据表**

|  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- |
| **参考限名称** | **编号** | **颜色** | **线条** | **粗细** | **描述** | **是否可删除** |
| 法规限 | REG\_LIMIT | #FF0000 | 实线 | 1px | 法规或注册标准规定的限值 | 否 |
| 内控限 | IC\_LIMIT | #0000FF | 实线 | 1px | 内部控制标准，严于法规限 | 否 |
| 警戒限 | WARN\_LIMIT | #FFA500 | 虚线 | 1px | 企业内部警戒线，超出需关注 | 否 |
| 纠偏限 | CORR\_LIMIT | #800080 | 虚线 | 1px | 超出需采取纠正措施 | 否 |

注：默认参考线可编辑（名称、编号、颜色、线条、粗细、描述），但不可删除。编辑后编号需保持全局唯一。
