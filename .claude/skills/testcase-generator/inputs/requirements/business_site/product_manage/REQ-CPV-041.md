## 原始需求内容

### 产品管理

【需求背景】

产品工艺管理模块是工艺数据的核心入口，需要统一管理产品基础信息（名称、编码、剂型、规格等），并实时展示每个产品关联的工艺数据数量（工序、操作、CPM、CPP、CQA），支持产品的新增、导入导出、编辑、删除及复制操作，为后续工序、参数维护提供产品维度的数据隔离与快速复用能力。

【需求描述】

|  |  |
| --- | --- |
| **名称** | **需求描述** |
| **菜单入口** | 首页菜单下面增加一个以及菜单，名称为：产品工艺管理 |
| **需求说明** | 1.**产品列表展示**：左侧产品列表展示全部产品，支持按产品名称、产品编码模糊搜索。 |
| **页面布局** | 1.左侧产品列表宽度30%，右侧为TAB页签区域宽度70%。 2.左侧顶部有搜索框（产品名称、编码）、三个操作按钮（新增、导入、导出）。 3.列表分页显示，每页30个（是否分页，前端可视情况而定）。每条产品行显示产品名称、编码、剂型、规格及各参数数量（数字带下划线或蓝色链接样式）。 4.操作栏包含“编辑”、“删除”、“复制”图标。 |
| **异常情况处理** | 1.新增/编辑时产品编码已存在 → 提示“产品编码已存在，请修改”。 |
| **后置处理功能** | 1.新增/编辑成功后，刷新产品列表并自动选中该产品，右侧TAB页加载对应工艺数据。 |

**新增/编辑产品字段表**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **字段名称** | **输入类型** | **是否必填** | **默认值** | **说明** |
| 产品名称 | 文本框 | 是 | 无 | 最多200字符，全局唯一 必填，唯一性、最大支持200个字符 |
| 产品编码 | 文本框 | 是 | 无 | 全局唯一，建议格式：PD-XXXX，编辑时不可修改 必填、唯一性、最大支持100个字符 |
| 剂型 | 下拉单选 | 是 | 无 | 选项：片剂、胶囊剂、颗粒剂、注射剂、口服液、软膏剂等（可配置） |

**导入检验流程如下**

导入产品管理，首先需要先下载模版，**文件类型**仅支持.xlsx和.xls格式的Excel文件，弹窗的文件选择器应默认筛选此类文件，其他文件类型默认不可见。

**下载Excel模版，鼠标悬停表头信息展示为：**

|  |  |  |
| --- | --- | --- |
| Excel模版表头字段信息 | 类型 | 鼠标悬停在表头上文字显示 |
| 产品名称 | 文本框 | 必填，唯一性、最大支持200个字符 |
| 产品编码 | 文本框 | 必填、唯一性、最大支持100个字符 |
| 剂型 | 下拉框 | 必填。选项范围为：片剂、胶囊剂、颗粒剂、注射剂、口服液、软膏剂等（可配置） |
| 规格 | 下拉框 | 必填项 |
| 生产商 | 文本框 | 必填 |
| 产品描述 | 文本框 | 非必填项 |

**以下是导入文件相关的流程整理：**

|  |  |  |  |
| --- | --- | --- | --- |
| 1 | 上传文件 | 用户上传Excel文件到系统。 | 文件上传至服务器待处理。 |
| 2 | 文件格式校验 | 系统检查文件格式（如扩展名、模板结构等）是否符合要求。 | - 若格式错误，直接提示“文件内容不匹配”/“请使用系统模版”，流程终止。 |
| 3 | 逐行数据校验 | 对每一行数据进行业务规则校验（如必填项、数据格式等），记录错误行号和错误原因。多个错误原因用分号隔开。 | 生成错误信息列表（包含行号及对应错误原因）。 |
| 4 | 判断是否有错误 | 检查校验过程中是否发现任何错误。 | -若无错误，转步骤5。 |
| 5 | 产品信息导入 | 将产品信息导入系统，导入完成后提示“XXXX个产品导入成功”。 | 显示成功提示，导入完成。 |
| 6 | 生成错误报告 | 整份文件存在一个以上错误时，不允许导入，并生成TXT格式错误报告。 报错内容格式： | 提供错误报告TXT文件下载，用户可获取详细失败记录。 |

**上传文件需要对文件文件格式进行校验：**

|  |  |  |
| --- | --- | --- |
| 序号 | 异常情况 | 对应提示 |
| 2 | Excel表格数据中存在合并单元格 | 文件中存在合并单元格，暂不支持。请处理后重新上传。 |
| 3 | 文件中有表头，但无数据，出对应提示 | 文件中无有效数据，请检查后重新上传 |
| 4 | 加密软件异常失败校验 | 比如加密软件等，导入时，提示：Excel解析失败，请检查格式等信息。 |
| 5 | 上传的文件未确定之前被删除或文件损坏 | 文件已不存在或格式错误，请检查文件。 |
| 6 | 表格中有数据，但表头为空 | 表头信息存在空值，暂不支持分析 |
| 7 | 导入一份空文件 | 文件为空，暂不支持导入 |
| 8 | 文件产品个数超过2000个 | 文件产品个数超过2000个 |
