# 界面设计图说明

## 存放内容

本目录用于存放淘宝项目的界面设计图、截图说明、交互说明和视觉标注文档。

## 阅读重点

读取界面设计图时，应重点提取：

- 页面入口
- 页面标题
- 输入框、按钮、弹窗、提示条等控件
- 默认状态、选中状态、禁用状态和错误状态
- 列表、卡片、价格、库存、优惠等展示字段
- 页面跳转关系
- 空状态、加载状态和失败状态

## 用例生成要求

界面设计图不仅用于验证页面展示，还应转化为交互测试用例。例如按钮是否可点击、禁用状态是否正确、错误提示是否展示、页面刷新后状态是否保留。

## 淘宝典型关注点

- 商品图片是否支持切换和预览。
- 规格选择后价格和库存是否变化。
- 购物车勾选状态是否影响金额。
- 结算页优惠券不可用原因是否展示。
