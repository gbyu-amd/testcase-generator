# 用例模板库说明

## 目录用途

本目录用于保存通用用例模板和各功能模块的参考用例。生成新用例时，应先参考通用模板、模块说明和已有用例，但不要修改本目录内容。

## 维护原则

- 通用模板只描述字段结构和编写方法。
- 功能模块目录保存只读参考用例。
- 新生成或补充的用例必须保存到 `outputs/origin_exports/<module_name>_testcases.md`。
- 不允许把多个模块的新用例混写到一个输出文件中。
- 生成前必须检查参考用例，避免重复场景。

## 固定字段

| 用例编号 | 功能模块 | 测试场景 | 优先级 | 前置条件 | 操作步骤 | 预期结果 | 用例类型 | 来源 |
|---|---|---|---|---|---|---|---|---|

## 用例类型

- 正向
- 异常
- 边界
- 权限
- 回归
- 兼容
- 联动
