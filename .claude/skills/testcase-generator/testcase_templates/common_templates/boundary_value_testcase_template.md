# 边界值用例模板

## 适用范围

适用于验证输入长度、数量范围、价格区间、库存数量、优惠门槛等边界场景。

## 编写要求

- 明确边界值来源。
- 至少覆盖最小值、最大值、低于最小值、高于最大值。
- 预期结果必须说明系统允许、拦截或提示的具体表现。

## 淘宝常见边界

- 搜索关键词为空、一个字符、超长字符。
- 购物车商品数量为一、库存上限、超过库存。
- 优惠券刚好满足门槛、低于门槛一分钱。
- 收货地址字段为空、超长、包含特殊字符。
