# 边界值用例模板

## 适用范围

适用于验证 CPV 系统中的输入长度、文件大小、批次数量、周期范围、阶段数、字段类型、数据范围、控制限、规格限、日期区间、审批意见长度等边界场景。

## 编写要求

- 明确边界值来源。
- 至少覆盖最小值、最大值、低于最小值、高于最大值。
- 预期结果必须说明系统允许、拦截或提示的具体表现。
- 涉及数据分析时，应说明分析结果、未分析状态、字段清理或错误提示是否符合规则。

## CPV 常见边界

- 产品编码、方案名称、报告名称、审批意见为空、最短、最长和超长。
- 年度计划周期起止日期相同、跨年、结束日期早于开始日期。
- 监控频率为最小值、最大值、低于最小值和高于最大值。
- 批次数量为 0、1、规则要求的最小批次数、最大批次数和超过上限。
- Excel 文件为空、刚好达到大小上限、超过大小上限、字段数量达到上限。
- 分析阶段数为 1、规则允许的最大阶段数、超过最大阶段数。
- 数值型字段等于规格下限、规格上限、低于下限、高于上限。
- 控制限、规格限、日期区间和小数精度达到规则边界。
