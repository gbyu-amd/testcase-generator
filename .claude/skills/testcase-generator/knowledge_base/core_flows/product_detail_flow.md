# 商品详情流程

## 流程目标

验证商品详情页可以准确展示商品信息，并支持规格选择、加入购物车、立即购买、收藏、评价查看等操作。

## 主流程

1. 用户从搜索结果、推荐位或购物车进入商品详情页。
2. 页面展示商品标题、图片、价格、销量、评价、店铺和服务信息。
3. 用户选择商品规格和购买数量。
4. 页面刷新规格对应的价格、库存和图片。
5. 用户点击加入购物车或立即购买。

## 分支流程

- 未选择必选规格时提示选择规格。
- 库存不足时禁止购买或限制数量。
- 商品下架时禁用购买按钮。
- 价格变化时刷新展示价格。
- 未登录点击加入购物车或立即购买时跳转登录。

## 测试重点

- 规格联动。
- 价格和库存刷新。
- 商品状态变化。
- 商品详情页与购物车、结算页金额一致。
