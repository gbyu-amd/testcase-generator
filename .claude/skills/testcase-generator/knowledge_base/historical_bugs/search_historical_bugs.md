# 搜索历史缺陷

## 缺陷一：搜索空结果未展示空状态

- 影响模块：搜索
- 严重程度：中
- 触发条件：用户搜索无任何匹配商品的关键词。
- 历史问题：页面停留在加载状态或展示空白，未给出空结果提示。
- 回归关注：无结果时应展示"没有找到相关商品"空状态，并提供修改关键词或推荐入口。

## 缺陷二：筛选与排序条件叠加后结果错误

- 影响模块：搜索
- 严重程度：高
- 触发条件：用户先按价格排序，再叠加品牌筛选。
- 历史问题：筛选生效后排序条件丢失，结果顺序错乱。
- 回归关注：多个筛选与排序条件叠加时应同时生效，且翻页后条件保持。

## 缺陷三：分页加载重复或丢失商品

- 影响模块：搜索
- 严重程度：中
- 触发条件：用户在搜索结果页快速连续上拉加载更多。
- 历史问题：同一商品在不同页重复出现，或部分商品被跳过。
- 回归关注：分页加载不应重复或遗漏商品，加载中需禁止重复触发。
